### 1.1 Nature, portée, contexte
- **Application** : Apex AI - assistant personnel intelligent (PWA iPhone)
- **Type** : SPA monolithique 2.9 MB (en cours refacto Phase 1+2 modules)
- **Architecture** : Local-first + Firebase sync admin + Anthropic API IA + ElevenLabs voice clone

### 1.2 Catégories de personnes concernées
- Admin Kevin DESARZENS (Monaco)
- Clients pré-configurés (Laurence et autres)

### 1.3 Catégories de données
- **Identité** : nom, prénom, email, téléphone, photo (RGPD Art. 4)
- **Conversations IA** : texte chat, images, fichiers uploadés
- **Biométrique** : voiceprint (`ax_voice_print_<uid>`), Face ID (PIN device)
- **Géolocalisation** : `ax_user_locations` (FB_LOCAL strict v12.535)
- **Bancaire** : IBAN, BTC/ETH addresses, PayPal handle (chiffré v12.529)
- **Santé** : `ax_health_reports` (vie privée, FB_LOCAL strict v12.535)
- **Comportement** : audit logs, métriques usage

## 2. Nécessité et proportionnalité

### 2.1 Base légale (Art. 6)
- **Art. 6.1.b** : Exécution contrat (assistant personnel demandé par utilisateur)
- **Art. 6.1.f** : Intérêt légitime (amélioration UX, sécurité)
- **Art. 9.2.a** : Consentement explicite pour données sensibles (biométrique, santé)

### 2.2 Minimisation
- Local-first : 95% données restent sur device (LS + IDB)
- Firebase sync : seulement admin Kevin cross-device (FB_FIX)
- Anthropic API : texte conversation seulement, pas de stockage Anthropic (ZDR)

### 2.3 Proportionnalité IA
- **EU AI Act classification** : Limited Risk (chatbot informationnel)
- Pas de scoring social, pas de surveillance biométrique temps réel public
- Voiceprint = opt-in user pour reconnaissance "Dis Apex"

## 4. Mesures techniques

### Chiffrement
- LS critical keys : AES-GCM 256 (PBKDF2 100k, salt fingerprint)
- Transport : TLS 1.3 partout
- Voiceprint : FB_LOCAL strict + chiffré per-user

### Auth
- PIN per-user (`ax_pin_<uid>`)
- Face ID / Touch ID via WebAuthn
- Rate-limit progressif v12.536 [0,0,0,0,0,30s,2min,10min,1h,24h]
- axIsAdminStrict double-check RAM/LS v12.530

### Audit
- `ax_audit` (max 500), `ax_security_log`, `ax_breach_log`
- `axBreachNotification()` 72h auto-escalation Kevin
