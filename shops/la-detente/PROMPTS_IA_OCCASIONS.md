# 🎨 Prompts IA — Modèles & Occasions (La Détente)

Prompts **prêts à coller** dans le studio IA de la boutique (worker Gemini) pour générer un **motif illustré original** (style aquarelle / dessin / fun) à associer aux modèles texte de `occasions.html`. Remplace `{PRÉNOM}` si tu veux l'intégrer dans l'image (sinon le texte est déjà géré par le modèle).

> Règle commune (à garder dans chaque prompt) : *« illustration, pas une photo, fond blanc pur, style aquarelle + crayon de couleur, couleurs vives, contour net, centré, isolé, sans texte parasite »*. Le rendu est **original** (à toi), aucune copie.

### Comment utiliser
1. Ouvre le **Studio** de la boutique (branche `claude/textile-shop-ar15-heart-mMJ0j`).
2. Colle le prompt de l'occasion → l'IA génère le **motif illustré original**.
3. Combine-le avec le **texte du modèle** (`occasions.html` : prénom/nom en direct) → pose sur le produit.

> ⚠️ Production : un visuel illustré marche sur tout produit DTG dans la **zone d'impression** ; pour du *edge-to-edge*, choisir un produit **All-Over Print (AOP)**.
