2. Sécurité de tes données

🛡 Chiffrement militaire 🔒 Serveur aveugle 🛰 Post-quantum (PQXDH)

Tes messages sont protégés par le chiffrement PQXDH (Signal Protocol post-quantum), le même standard utilisé par Signal et iMessage. Le serveur ne peut jamais lire le contenu de tes messages — il transporte uniquement des données chiffrées qu'il ne peut pas déchiffrer.

  • Chiffrement bout-en-bout par défaut sur tous les messages, photos, vidéos, fichiers, appels
  • Clés privées stockées uniquement sur ton téléphone (jamais envoyées ailleurs)
  • Sauvegarde chiffrée optionnelle vers iCloud/Google Drive (clé locale uniquement)
  • FaceID/Touch ID pour ouvrir l'app
  • Mode "Disparition" : messages auto-supprimés après délai choisi
