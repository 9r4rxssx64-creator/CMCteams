Outil pour debloquer une PWA iPhone bloquee sur ancienne version

Que fait ce bouton ?

1. Desinscrit tous les Service Workers Apex
2. Efface TOUS les caches PWA
3. Garde tes donnees user (localStorage intact)
4. Recharge Apex en version la plus recente

Si ca ne suffit pas

Sur iPhone, une PWA installee garde parfois un cache extrement tenace. Solution definitive :

  • 1. Reglages iPhone
  • 2. Safari
  • 3. Avance > Donnees de sites web
  • 4. Cherche "github.io" ou "cmcteams"
  • 5. Glisse a gauche > Supprimer
  • 6. Reinstalle Apex via Safari (icone partage > Sur l ecran d accueil)
