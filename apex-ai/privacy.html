3. Destinataires (sous-traitants RGPD Art.28)

  • Anthropic PBC (USA, via Cloudflare Worker UE) — modèles Claude Haiku/Sonnet/Opus 4.x : tes prompts texte/images sont envoyés à api.anthropic.com pour générer les réponses IA. Pas de stockage persistant côté Anthropic (ZDR commercial). DPA officiel : anthropic.com/legal/dpa · Privacy : anthropic.com/privacy
  • ElevenLabs Inc. (USA, optionnel) — synthèse vocale haute qualité : si tu actives la voix premium, le texte à lire est envoyé à ElevenLabs pour générer l'audio. Voiceprint utilisateur (clone vocal "Dis Apex") reste LOCAL strict (jamais transmis). DPA : elevenlabs.io/dpa · Privacy : elevenlabs.io/privacy
  • Firebase / Google LLC (UE, region europe-west1) : synchronisation cross-device chiffrée AES-256 au repos + TLS 1.3 transport. Tes données n'entrent pas dans Google Ads. DPA : firebase.google.com/terms/data-processing-terms
  • Stripe Payments Europe Ltd. (Irlande, UE) — paiements abonnements. PCI-DSS Level 1. stripe.com/privacy
  • Cloudflare Inc. (USA + edge UE) — proxy Worker pour CORS + masquage IP utilisateur. Pas de stockage durable des prompts. cloudflare.com/privacypolicy
  • GitHub Pages / Microsoft (USA) — hébergement statique HTML/CSS/JS de l'app. Aucune donnée utilisateur transitée.
  • OpenRouter / Groq / Google Gemini (failover IA optionnel, USA) — utilisés UNIQUEMENT si Anthropic indisponible. Activable/désactivable dans Réglages.

4. Transferts hors UE

Les transferts vers les USA (Anthropic, GitHub, Stripe) sont couverts par les clauses contractuelles types de l'UE + Data Privacy Framework (USA). Monaco adhère aux décisions d'adéquation UE.

5. Sécurité

  • HTTPS obligatoire sur toute l'app
  • Tokens stockés localStorage (jamais en URL / logs)
  • PIN hashé (pas en clair)
  • Option Face ID / Touch ID (WebAuthn / PassKeys)
  • Session TTL 24h (30 jours sur appareil de confiance)
  • Audit Bodyguard : log chaque connexion avec IP + fingerprint
  • Firebase rules restrictives (lecture/écriture authentifiée uniquement)
  • Kill switch global si urgence (tu coupes tout en 1 clic)

7. Cookies / Tracking

Apex AI n'utilise aucun cookie tiers (pas de Google Analytics, pas de publicité). Seul le localStorage navigateur est utilisé pour le fonctionnement de l'app — c'est techniquement requis pour la PWA, pas un tracker. Pas de bandeau cookies obligatoire.

8. Intelligence Artificielle (EU AI Act — Règlement 2024/1689)

Apex AI utilise des modèles de langue de grande taille (LLM) Claude d'Anthropic. Conformément à l'Article 52 EU AI Act :

  • Classification : Système IA à risque limité (chatbot informationnel, pas d'évaluation biométrique temps réel public, pas de scoring social).
  • Disclosure obligatoire : toute réponse générée par IA est marquée par un badge 🤖 IA dans l'interface chat.
  • Modèles utilisés : Claude Haiku 4.5 (rapide), Claude Sonnet 4.6 (équilibré), Claude Opus 4.7 (raisonnement profond). Sélectionnable dans Réglages.
  • Données d'entraînement : Apex n'entraîne aucun modèle. Les modèles utilisés sont ceux d'Anthropic (entraînement public + RLHF + Constitutional AI).
  • Limites connues : hallucinations possibles, biais culturels (entraînement majoritairement anglophone), pas de connaissance temps réel sans web search activé.
  • Décision automatisée significative : Apex ne prend AUCUNE décision automatisée à effet juridique sur l'utilisateur (pas de scoring crédit, pas de filtrage emploi). Art.22 RGPD non applicable.
  • Droit de contestation : tu peux à tout moment refuser une réponse IA et demander révision humaine via support@apex-ai.fr.
