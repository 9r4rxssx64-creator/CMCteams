3. Destinataires

  • Anthropic (USA, via Cloudflare Worker UE) : tes prompts sont envoyés pour générer les réponses IA. Anthropic applique sa propre politique : anthropic.com/privacy
  • Firebase / Google (UE, europe-west1) : synchronisation cross-device chiffrée. Tes données n'entrent pas dans Google Ads.
  • Stripe (UE) : paiements. stripe.com/privacy
  • Cloudflare (UE) : proxy API. Pas de stockage durable.
  • GitHub Pages (USA) : hébergement statique de l'app.

4. Transferts hors UE

Les transferts vers les USA (Anthropic, GitHub, Stripe) sont couverts par les clauses contractuelles types de l'UE + Data Privacy Framework (USA). Monaco adhère aux décisions d'adéquation UE.

5. Sécurité

  • HTTPS obligatoire sur toute l'app
  • Tokens stockés localStorage (jamais en URL / logs)
  • PIN hashé (pas en clair)
  • Option Face ID / Touch ID (WebAuthn / PassKeys)
  • Session TTL 24h (30 jours sur appareil de confiance)
  • Audit Bodyguard : log chaque connexion avec IP + fingerprint
  • Firebase rules restrictives (lecture/écriture authentifiée uniquement)
  • Kill switch global si urgence (tu coupes tout en 1 clic)

6. Tes droits RGPD

Tu peux à tout moment exercer les droits suivants :

  • Accès : export complet JSON (Réglages > Mes données > Télécharger)
  • Rectification : modifier ton profil directement dans l'app
  • Effacement : bouton "Supprimer toutes mes données" dans Réglages
  • Portabilité : export JSON exploitable ailleurs
  • Opposition / Limitation : désactiver features via toggles (géoloc, micro, caméra)
  • Plainte : CNIL (France) ou APDP (Monaco) si tu considères tes droits violés

Pour exercer ces droits : support@apex-ai.fr — réponse sous 30 jours.

7. Cookies / Tracking

Apex AI n'utilise aucun cookie tiers (pas de Google Analytics, pas de publicité). Seul le localStorage navigateur est utilisé pour le fonctionnement de l'app — c'est techniquement requis pour la PWA, pas un tracker. Pas de bandeau cookies obligatoire.

8. Modifications

Cette politique peut être modifiée. Les utilisateurs sont notifiés 30 jours avant toute modification substantielle.
