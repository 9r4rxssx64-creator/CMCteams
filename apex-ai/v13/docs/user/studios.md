# Les 10 Studios créatifs Apex

Apex propose 10 studios pour créer du contenu pro/perso au top niveau.

## 1. 🎚 Studio Music

**Mixage musique pro** style table de mixage.

- 12+ pistes simultanées
- EQ multi-bandes (graves/médiums/aigus)
- Effets : reverb, delay, chorus, flanger, phaser
- Compresseur master multi-band
- Limiter LUFS (broadcast standard)
- Noise gate, auto-tune, sidechain
- Stem separation (séparer voix/instruments)
- Export : WAV 24-bit, MP3 320kbps, FLAC
- BPM auto-détection + sync

**Cas d'usage** : DJ, producteur, podcaster, beatmaker, créateur YouTube.

## 2. 🎬 Studio Video

**Montage vidéo** style CapCut.

- Timeline multi-pistes
- Cuts, fade in/out, transitions
- Captions auto (transcription IA + traduction)
- Filtres et grading couleur
- Audio multi-piste
- Templates TikTok / Reels / YouTube Shorts
- Export : MP4 1080p, 4K, vertical/horizontal

**Cas d'usage** : créateur de contenu, vlog, TikTok, pub, montage souvenirs.

## 3. 📄 Studio CV

**Génération de CV** moderne.

- 12+ templates professionnels
- Auto-rempli depuis ton profil Apex
- Sections : expérience, formation, compétences, langues, hobbies
- Optimisation ATS (passe les filtres recruteurs)
- Export PDF, Word, version web
- Lettre de motivation IA générée

**Cas d'usage** : recherche emploi, freelance, portfolio.

## 4. 🧾 Studio Facture

**Facturation pro** complète.

- Templates France + Monaco + EU
- Mention TVA automatique (taux selon pays)
- Numérotation auto séquentielle
- Conditions paiement (30j, 60j, comptant)
- Méthodes : virement (IBAN), Stripe, PayPal, crypto
- Relances auto (J+15, J+30, J+45)
- Export PDF + envoi email
- Suivi paiements

**Cas d'usage** : freelance, indépendant, micro-entreprise.

## 5. 📑 Studio Contrat

**Contrats juridiques** France/Monaco.

- 40+ templates : CDI, CDD, NDA, prestation, location, vente
- Adaptation automatique selon parties
- Articles légaux à jour (Code travail, Code civil)
- Signature électronique (Yousign, DocuSign API)
- Stockage chiffré
- Versions multilingues (FR/EN/IT)

**Cas d'usage** : recrutement, partenariat, freelance, location.

## 6. 🎤 Studio Présentation

**Slides pro** style Keynote/Pitch.

- 20+ thèmes premium
- Génération automatique depuis prompt
- Charts et graphiques dynamiques
- Animations de transition
- Mode présentateur
- Export : PDF, PPTX, vidéo MP4
- Speaker notes générées

**Cas d'usage** : pitch startup, présentation client, formation.

## 7. 🎨 Studio Logo

**Création de logo** assistée IA.

- Brief en langage naturel ("logo restaurant italien moderne")
- 12+ variantes générées
- Vectoriel SVG
- Palette couleurs cohérente
- Typographie pro
- Variations : light/dark, horizontal/vertical, monogramme
- Export : SVG, PNG (transparent), ICO favicon

**Cas d'usage** : entrepreneur, refresh marque, projet perso.

## 8. 🏗 Studio Architecture

**Plans et calculs bâtiment**.

- Calcul de surface (m², pi²)
- Mélanges béton/peinture/chaux/carrelage
- Normes : RE2020, RT, DTU
- Accessibilité PMR/ERP
- Calcul Blondel (escalier)
- Palette Pantone 2026
- Plan 2D simple
- Comparatif matériaux

**Cas d'usage** : auto-rénovation, bricolage, projet construction.

## 9. 📋 Studio Préfecture

**Démarches administratives**.

- Dossiers : titre séjour, naturalisation, CNI, passeport
- Checklist documents requis
- Templates lettres officielles
- Liens directs ANTS, service-public.fr
- Calendrier rendez-vous
- Notification rappel pré-RDV
- Suivi dossier

**Cas d'usage** : démarches France, démarches Monaco, étrangers.

## 10. 📷 Studio Photo / Clip

**Retouche photo + clip vidéo depuis photo**.

- Filtres Instagram-like
- Recadrage intelligent
- Suppression background (IA)
- Upscale 4× (super-résolution)
- Animation photo → vidéo (Ken Burns, parallax)
- Slideshow musical
- Export : JPG, PNG, MP4

**Cas d'usage** : retouche perso, créateur contenu, événement.
