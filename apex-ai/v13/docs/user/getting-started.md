# Premier contact avec Apex AI — Guide en 5 minutes

Bienvenue ! Ce guide t'aide à prendre en main Apex AI rapidement.

## 1. Création de compte (30 secondes)

1. Ouvre Apex AI dans ton navigateur (Safari iPhone recommandé pour PWA).
2. Saisis ton **prénom + nom**.
3. Choisis un **PIN à 4-6 chiffres** (sécurisé : pas 1234 !).
4. Confirme ton PIN.
5. ✅ Compte créé !

> 💡 **Astuce** : Active FaceID au premier login pour ne plus jamais retaper ton PIN.

## 2. Installer en application (PWA — 20 secondes)

### Sur iPhone (Safari)
1. Touche le bouton **Partager** ↑ en bas.
2. Descends et touche **"Ajouter à l'écran d'accueil"**.
3. Renomme si tu veux, puis **"Ajouter"**.
4. ✅ Apex est sur ton écran d'accueil comme une vraie app !

### Sur Android (Chrome)
1. Touche le menu **⋮** en haut à droite.
2. Choisis **"Installer l'application"**.
3. Confirme.
4. ✅ Apex est installé !

## 3. Ton premier message (30 secondes)

1. Tu arrives sur l'écran de chat.
2. Apex te dit "Bonjour [Prénom], qu'est-ce que je peux faire pour toi ?"
3. Tape ta question (ou touche le **🎙 micro** pour dicter).
4. Apex répond en temps réel (effet streaming).

## 4. Découvrir les outils (1 minute)

Au bas de l'écran, 4 raccourcis :

- 🎨 **Modèles de projet** : 12+ thèmes (musique, vidéo, archi, juridique...)
- 🎓 **Tour découverte** : tutoriel guidé étape par étape
- ✨ **Mes compétences IA** : ce qu'Apex peut faire pour toi
- 📒 **Mon bloc-notes** : projets, favoris, notes

## 5. Activer la reconnaissance vocale (option, 30 secondes)

Si tu veux que Apex te reconnaisse à la voix :

1. Va dans **Réglages → Voix**.
2. Touche **"Enrôler ma voix"**.
3. Dis 3 fois : *"Apex, tu reconnais ma voix maintenant"*.
4. ✅ Maintenant tu peux dire **"Dis Apex"** depuis n'importe où dans l'app !

## 6. Tes premières fonctionnalités à essayer

- 🎵 Dis : **"Mixe-moi une musique"** → ouvre le Studio Mix
- 🎬 Dis : **"Fais un montage vidéo"** → ouvre le Studio Vidéo
- 🏗 Dis : **"Plan de cuisine 12m²"** → ouvre Architecture
- 🌐 Dis : **"Va sur Google"** → ouvre le navigateur intégré
- 🍳 Dis : **"Recette poulet curry"** → ouvre Cuisine Pro
- 💰 Dis : **"Calcule mon impôt 2026"** → ouvre Finance Pro

## 7. Conversations sauvegardées

Toutes tes conversations sont automatiquement sauvegardées dans la **sidebar gauche** avec un nom auto-généré (ex: "🎵 Mix samedi", "🏗 Réno cuisine").

Touche une conversation pour la reprendre.

## 8. Sécurité — bonnes pratiques

- ✅ Active FaceID/TouchID
- ✅ Active la **reconnaissance vocale** pour 2FA
- ✅ Sauvegarde tes clés API dans le **Coffre** (chiffré AES-GCM 256)
- ❌ Ne donne JAMAIS ton PIN à quelqu'un
- ❌ Ne stocke JAMAIS de seed phrase crypto dans Apex (utilise un hardware wallet)
