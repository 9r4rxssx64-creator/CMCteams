# Les 8 Modules Pro Apex

Modules d'expertise niveau professionnel pour les domaines spécialisés.

## 1. 📒 Légal Pro

**Droit français + jurisprudence + Monaco**

- 25+ codes français (Civil, Pénal, Travail, Commerce, Consommation...)
- Jurisprudence : Cassation, Conseil d'État, CJUE, CEDH
- Constitution Monaco
- 40+ templates lettres juridiques
- Calculs : prescription, indemnités licenciement
- Procédures pas-à-pas
- Liens Légifrance, Légimonaco

**Avertissement** : Information indicative. Consulte un avocat pour décision importante.

## 2. 💰 Finance / Fiscalité Pro

**Calculs fiscaux + Monaco**

- IR France 2026 (tranches, quotient familial)
- PFU 30% (capital, dividendes)
- Plus-values immobilières (abattements 22 ans / 30 ans)
- Plus-values mobilières
- Crédit immobilier (mensualités, capacité, frais notaires)
- Régime fiscal Monaco
- Comparatif statuts (auto-entrepreneur, EURL, SASU)
- Convention fiscale FR-Monaco

**Avertissement** : Consulte un expert-comptable pour ta situation précise.

## 3. 💊 Médical Pro

**Vidal + IMC + urgences**

- Vidal : médicaments OTC + posologies
- Interactions médicamenteuses
- IMC + métabolisme basal
- Calorimétrie + besoins nutritionnels
- Urgences SAMU (15) + numéros utiles
- Vaccins (calendrier 2026)
- Premiers secours (gestes essentiels)

**Avertissement** : Information générale. JAMAIS substitut consultation médicale.

## 4. 🍳 Cuisine Pro

**50+ recettes + cuissons + allergènes**

- Recettes FR + internationales (italien, asiatique, oriental)
- 30+ cuissons précises (températures, temps)
- 14 allergènes INCO (gluten, lactose, fruits coque...)
- Calories / macros / vitamines / index glycémique
- Plans menus 7 jours (équilibrés)
- Liste de courses auto
- Substitutions diététiques (végé, kéto, paléo, sans gluten, halal, kasher)
- Sommellerie (accord vins)
- Modes : keto, paléo, méditerranéen, DASH, IF (jeûne intermittent)

**Cas d'usage** : famille, régime, batch cooking, événement.

## 5. 🌐 Traducteur Pro

**30 langues + interprète temps réel**

- Traduction texte (DeepL via Claude Haiku)
- 30+ langues (toutes langues EU + japonais, coréen, chinois, arabe, hébreu, russe...)
- Mode interprète : voix → voix temps réel
- Cache (économise crédits)
- Détection auto langue source
- Glossaire métier (juridique, médical, technique)
- Adaptations culturelles (formules politesse)

**Cas d'usage** : voyage, business international, traduction docs.

**Outils entrepreneuriat**

## 8. 🎓 Education Pro

**Outils apprentissage + parents**

- Aide aux devoirs (toutes matières, tous niveaux)
- Explications pas-à-pas
- Génération d'exercices
- Correction rédaction (français, langues)
- Méthodes d'apprentissage (Pomodoro, Cornell, etc.)

**Cas d'usage** : parents, étudiants, formateurs, profs particuliers.
