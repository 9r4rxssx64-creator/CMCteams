## 4) ⚙️ Activer les catalogues LIVE (worker proxy)

Le catalogue auto a besoin d'un petit worker `kdmc-sourcing-proxy` qui détient les clés (côté serveur, jamais dans le navigateur). Fichier prêt : `services/sourcing-proxy/worker.js` (adaptateurs Printful + Printify livrés, autres à brancher).

**Quand tu auras des clés** (ex. Printful) :
1. Déploie le worker `kdmc-sourcing-proxy` (Cloudflare).
2. Ajoute le secret : `PRINTFUL_TOKEN` (et/ou `PRINTIFY_TOKEN`).
3. Dans l'app, fiche fournisseur → **🔌 Brancher le catalogue LIVE** → colle l'URL du worker → le vrai catalogue s'affiche.

> Sans worker/clé, l'app reste 100 % utilisable : bouton **« 🔗 Catalogue complet »** (ouvre le vrai catalogue du fournisseur) + **« 📝 Ajouter un produit manuellement »** → la sélection partagée Kevin+Laurence + export CSV fonctionnent déjà.

## 5) 🧭 Rappel — comment ça marche

1. Va sur la **famille** voulue (section 2) → choisis un **fournisseur** → ouvre son **catalogue**.
2. Clique **✅ Mettre en vente** (live) ou **📝 Ajouter manuellement** (catalogue externe).
3. La **sélection** est partagée temps réel **Kevin + Laurence** → **⬇️ Export CSV** pour la pousser en boutique.
4. Tu **supprimes dans l'app** les fournisseurs que tu ne gardes pas.
