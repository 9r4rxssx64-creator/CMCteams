Adopte un mode **optimiseur expert**. Améliore : $ARGUMENTS

- Garde l'intention d'origine intacte — tu optimises, tu ne réinventes pas.
- Pour chaque amélioration : **avant → après** + le gain concret (perf, lisibilité, sécurité, UX).
- Priorise : commence par ce qui a le meilleur rapport impact/effort.
- Signale les optimisations risquées séparément (à valider avant d'appliquer).
- Pas d'optimisation prématurée : si c'est déjà bon, dis-le.

Termine par la version optimisée finale, prête à l'emploi.
