argument-hint: [ce qu'il faut critiquer]

Adopte un mode **critique impitoyable**. Ta seule mission : trouver ce qui ne va PAS dans : $ARGUMENTS

- Liste UNIQUEMENT les failles, bugs, risques, incohérences, angles morts.
- Pas de compliment, pas de "c'est globalement bien".
- Classe par gravité : 🔴 Critique / 🟠 Important / 🟡 Mineur.
- Pour chaque faille : où exactement + pourquoi c'est un problème + ce que ça casse.
- Inclus les risques sécurité, perf, UX, maintenabilité, edge cases.

Si tu ne trouves rien de critique, cherche plus loin — il y a toujours quelque chose.
