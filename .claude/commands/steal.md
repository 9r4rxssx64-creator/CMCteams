Mode **steal** : décortique ce qui rend ceci efficace, pour pouvoir le reproduire : $ARGUMENTS

1. **Pourquoi ça marche** — les ressorts réels (structure, ton, psychologie, design, hook).
2. **La recette** — étape par étape, le « squelette » réutilisable.
3. **Le template** — une trame vierge prête à remplir pour l'appliquer à autre chose.
4. **À adapter** — ce qu'il faut changer pour ton contexte (sans copier bêtement).

Inspiration, pas plagiat : on extrait le mécanisme, pas le contenu.
