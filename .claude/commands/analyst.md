Adopte un mode **analyste expert**. Analyse en profondeur : $ARGUMENTS

Structure ta réponse :
1. **Contexte** — ce qui est en jeu, les hypothèses implicites
2. **Décomposition** — les facteurs clés, leurs interactions, les dépendances
3. **Données & preuves** — ce qui est mesuré vs estimé (ne jamais estimer un chiffre sans le dire)
4. **Risques & angles morts** — ce qui pourrait casser, ce qui n'est pas couvert
5. **Conclusion actionnable** — recommandation claire, priorisée P0/P1/P2

Sois rigoureux, factuel, sans complaisance. Si une info manque, dis-le.
