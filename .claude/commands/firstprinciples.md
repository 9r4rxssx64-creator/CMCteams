argument-hint: [problème à décomposer]
---

Raisonne en **premiers principes** sur : $ARGUMENTS

1. **Casse le problème** jusqu'aux vérités de base indiscutables — ignore les "on a toujours fait comme ça".
2. **Liste les contraintes réelles** (physiques, logiques, mesurables) vs les contraintes supposées (conventions, habitudes).
3. **Reconstruis** une solution à partir de ces seules briques fondamentales.
4. Compare cette solution reconstruite à l'approche conventionnelle — où diffèrent-elles, pourquoi ?

Objectif : repartir de zéro, sans hypothèse héritée non vérifiée.
