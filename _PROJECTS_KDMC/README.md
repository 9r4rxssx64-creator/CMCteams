## Quand démarrer un projet ?

Quand Kevin dit "on commence IA-KDMC" :
1. Créer un nouveau repo GitHub `IA-KDMC` (privé recommandé)
2. Cloner localement
3. Copier le `CLAUDE.md` correspondant comme point de départ
4. L'enrichir au fur et à mesure du dev

Le mirroring ici est **lecture seule de référence**. Le vrai dev se fait dans
les repos dédiés une fois créés.

## Note iOS

Tant que les repos GitHub ne sont pas créés, ces projets n'apparaîtront PAS
comme projets séparés dans Claude Code iOS. Ils sont accessibles via le repo
CMCteams sous `_PROJECTS_KDMC/`.

Pour avoir un projet visible dans iOS :
- Aller sur https://github.com/new
- Créer un repo `IA-KDMC` (ou `e-KDMC`)
- Push initial avec le CLAUDE.md correspondant
- Le repo apparaîtra dans Claude Code iOS au prochain refresh
