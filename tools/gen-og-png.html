Generateur OG Image PNG — APEX AI

Outil 1-clic pour generer og-image.png (1200×630) depuis le SVG actuel d'APEX. Telecharge le PNG, place-le dans apex-ai/og-image.png, puis update les meta og:image.

Utilisation :
1. Clique Generer + Telecharger
2. Le PNG telecharge s'appelle og-image.png
3. Place-le dans apex-ai/ (remplace l'SVG)
4. Modifie le meta og:image dans index.html : og-image.svg -> og-image.png
