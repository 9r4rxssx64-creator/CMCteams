🔗 Apex iOS Bridge — Déploiement 1-clic

v13.3.58 · Cloudflare Worker pour débloquer Bluetooth/IR/IoT depuis iOS Safari

Pourquoi ce Worker ?

iOS Safari bloque Web Bluetooth, Web NFC, Web USB. Ce Worker tourne sur Cloudflare edge et permet à Apex iOS de piloter Broadlink (TV/IR), eWeLink (Sonoff), Tuya, Hue et HomeKit via HTTPS.

Étapes

  1. Compte Cloudflare gratuit

    Si tu n'en as pas, crée un compte sur dash.cloudflare.com/sign-up (gratuit, 100 000 requêtes/jour).

  2. Créer le Worker

    Va sur dash.cloudflare.com → Workers → Create et nomme-le apex-ios-bridge.

  3. Coller le code Worker

    Copie tout le code de tools/apex-ios-bridge-worker.js et colle-le dans l'éditeur Cloudflare Worker.

  4. Configurer les variables d'environnement

    Dans le Worker → Settings → Variables, ajoute :

    APEX_BRIDGE_TOKEN     = (généré ci-dessous, à coller aussi dans Apex Coffre)
BROADLINK_TOKEN       = (token Broadlink Cloud, optionnel)
EWELINK_TOKEN         = (token eWeLink, optionnel)
TUYA_TOKEN            = (token Tuya, optionnel)
HUE_BRIDGE_URL        = (URL Hue Bridge LAN, optionnel)
HUE_USERNAME          = (username Hue, optionnel)
ALLOWED_ORIGINS       = https://9r4rxssx64-creator.github.io

  5. Déployer

    Clique sur Save and Deploy dans le Worker. Tu obtiens une URL : https://apex-ios-bridge.<ton-compte>.workers.dev

  6. Coller dans Apex Coffre

    Ouvre Apex iOS → Coffre → ajoute deux entrées :

    ax_apex_bridge_worker_url = https://apex-ios-bridge.<ton-compte>.workers.dev
ax_apex_bridge_token       = (le token généré ci-dessus)

  7. Tester depuis Apex

    Dans Apex iOS, ouvre le chat et tape : Liste mes devices Broadlink ou Allume la TV Clayton. Apex utilisera maintenant le Worker au lieu de fetch direct (qui était bloqué CORS sur iOS).

📊 Endpoints disponibles

Tous protégés par Authorization: Bearer <token> + CORS strict.

POST /broadlink/ir       Send IR command (TV, climatiseur, etc.)
POST /broadlink/learn    Apprendre code IR depuis télécommande
POST /ewelink/cmd        Sonoff/eWeLink device command
POST /tuya/cmd           Tuya Smart device command
POST /hue/lan            Philips Hue Bridge LAN
POST /homekit/scene      HomeKit scene trigger
GET  /scan-lan           Server-side LAN scan (mDNS)
GET  /health             Health check (no auth)

🔒 Sécurité

  • Auth Bearer token obligatoire
  • CORS strict (seul ton domaine Apex autorisé)
  • Rate limit 60 req/min par IP
  • Audit log immutable (KV namespace optionnelle)
  • Token rotation 30j recommandée
