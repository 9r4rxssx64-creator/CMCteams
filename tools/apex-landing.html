Questions fréquentes

Tout ce que tu dois savoir avant de commencer.

Mes données sont-elles privées ?

Oui. Apex est local-first : tout est stocké sur ton appareil par défaut. Le coffre est chiffré AES-GCM 256 avec ta passphrase. Firebase est optionnel pour la sync cross-device — tu peux désactiver.

Faut-il une clé API ?

Oui pour utiliser tes propres providers IA (Anthropic, OpenAI, etc.). Apex te guide pour les créer. Plan Pro inclut un quota partagé sans clé requise.

Compatible iPhone ?

Oui, Apex est une PWA installable sur iPhone (Safari ≥ 16.4). Mêmes capacités que sur desktop : voix, dictée, NFC, scan QR, push notifications.

Comment annuler mon abonnement ?

1 clic dans Réglages → Abonnement → Annuler. Effet immédiat. Tes données restent intactes.

RGPD ?

Apex est RGPD compliant. Tu peux exporter toutes tes données en JSON via Réglages → RGPD → Export. Suppression compte = effacement total local + cloud sous 24h.

Open source ?

Le code complet est sur GitHub : github.com/9r4rxssx64-creator/cmcteams. Audit communautaire bienvenu.
