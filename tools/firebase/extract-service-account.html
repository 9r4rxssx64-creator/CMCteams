Lit ton fichier .json Firebase et extrait les 2 valeurs à coller dans GitHub Secrets.

🔒 100 % local — rien n'est envoyé sur Internet

📂 1. Sélectionne ton fichier

Le fichier ressemble à cmcteams-c16ab-firebase-adminsdk-XXXXX-YYYYYYYY.json (téléchargé depuis Firebase Console).

🔐 2. FIREBASE_PRIVATE_KEY

📧 3. FIREBASE_CLIENT_EMAIL

À coller dans GitHub Secret nommé FIREBASE_CLIENT_EMAIL :

🎯 4. Va sur GitHub Secrets

  1. Ouvre : github.com/.../secrets/actions
  2. Pour chaque clé :
    • Si elle existe → tape dessus → Update secret → vide le champ → colle → Update
    • Si elle n'existe pas → New repository secret → Name : la clé → Value : colle → Add
  3. Re-déclenche le déploiement : Actions → Deploy apex-auth-worker → Run workflow

🔥 5. Sécurité — nettoyage

Quand le déploiement a réussi : retourne dans Fichiers → Téléchargements → supprime le fichier .json. Et tape le bouton ci-dessous pour effacer les valeurs de cette page.
