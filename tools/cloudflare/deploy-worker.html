Déployeur Push Worker — 1 clic
Remplis UNE SEULE des 3 cases ci-dessous selon ce que tu as.
L'outil prendra automatiquement la bonne.
🔑 Choix 1 — Token API RECOMMANDÉ
Le jeton scoped que tu viens de créer "Modifier Cloudflare Workers".
Source : Cloudflare → Jetons d'API
👁
💡 C'est le choix sécurisé : le token n'a accès QU'aux Workers, rien d'autre.
🌍 Choix 2 — Global API Key SI PAS CRÉÉ DE TOKEN
Ta clé legacy avec accès TOTAL au compte Cloudflare.
Source : Cloudflare → Clés API → "Affichage" sur Global API Key
👁
⚠️ Cette clé donne accès à TOUT ton compte. À utiliser uniquement si tu n'as pas de Token API.
🛡 Choix 3 — Origin CA Key PAS UTILE ICI
Cette clé sert à signer des certificats SSL Cloudflare → ton serveur.
Elle ne permet PAS de déployer un Worker. Tu peux l'ignorer pour cet outil.
📋 Choisis ton compte
🚀 Déployer le code Apex Push Worker
Le code apex-push-worker.js sera téléchargé depuis GitHub et déployé sur ton Worker apex-push-worker.
Tes 5 variables d'env (VAPID_*, FIREBASE_URL, ADMIN_TOKEN) sont préservées.
