Déployeur Push Worker — 1 clic
Étape 1 — Authentification Cloudflare
Choix 1 (recommandé) : Token API moderne
→ Créer un nouveau Token
→ Template "Edit Cloudflare Workers" → Create.

Choix 2 : Réutiliser ta Global API Key
→ Section "Clés API"
→ Tape "Affichage" sur Global API Key → Confirme avec ton mdp.
→ Tape sur "🌍 Global API Key" ci-dessous + colle email + clé.
Étape 2 — Choisis ton compte
Si plusieurs comptes, tape sur le bon.
Étape 3 — Déployer le code
Le code apex-push-worker.js sera téléchargé depuis GitHub et déployé sur ton Worker apex-push-worker.
Tes 5 variables d'env (VAPID_*, FIREBASE_URL, ADMIN_TOKEN) sont déjà configurées — elles seront préservées.
