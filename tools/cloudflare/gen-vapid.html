Generer cles VAPID

Outil pour creer les 2 cles cryptographiques du serveur push Apex.
Tout se passe dans ton navigateur, rien n'est envoye en ligne.
Cle PUBLIQUE
Cle PRIVEE
Important : sauvegarde la cle PRIVEE dans iCloud Notes / 1Password. Si tu la perds, il faudra regenerer + reconfigurer tout le monde. NE JAMAIS la partager.
OK : la cle PUBLIQUE peut etre partagee, c'est elle qui sera dans le code de Apex.

Que faire avec ces cles ?

1
Va dans Cloudflare Workers, ton worker apex-push-worker > Settings > Variables.
2
Ajoute VAPID_PUBLIC_KEY = ta cle publique (Encrypt OUI).
3
Ajoute VAPID_PRIVATE_KEY = ta cle privee (Encrypt OUI).
4
Dans Apex, code source apex-ai/index.html, remplace la ligne var AX_VAPID_PUBLIC=... par ta cle publique. Demande a Claude Code de le faire si besoin.
5
Voir DEPLOY-PUSH-WORKER.md pour les autres etapes.
